## 🎯 What is This?

The **Engineering Prompt Generator** is your ultimate companion for creating structured, professional learning prompts on any engineering topic. Whether you're preparing for technical interviews, studying complex concepts, or onboarding to new technologies, this tool generates comprehensive prompts that guide you from fundamentals to expert-level understanding.

### ✨ Why Use This Tool?

- 🧠 **Structured Learning**: Get prompts that follow proven educational frameworks
- ⚡ **Instant Generation**: No more staring at blank pages wondering where to start
- 🎯 **Interview Ready**: Perfect for technical interview preparation
- 📚 **Academic Excellence**: Ideal for deep-dive learning sessions
- 🏭 **Production Focused**: Real-world examples and best practices included

## 🎮 How to Use

### 1. **Enter Your Topic**
```
Type any engineering topic like:
• JavaScript Closures
• React Hooks
• Docker Containers
• REST API Design
• Database Indexing
• Microservices Architecture
```

### 2. **Generate & Customize**
- Click **"Generate Prompt"** to create your structured prompt
- The tool automatically replaces placeholders with your topic
- Get a comprehensive 9-section learning guide

### 3. **Copy & Use**
- Click **"Copy Prompt"** to copy to clipboard
- Use with any AI assistant (ChatGPT, Claude, etc.)
- Perfect for interview prep or deep learning sessions

## 📋 Generated Prompt Structure

Each generated prompt includes these 9 comprehensive sections:

### 🚀 **1. Introduction**
- Importance and relevance in real-world development
- Why this topic matters in modern engineering

### 🧠 **2. Core Concepts**
- Key terms and foundational principles
- Simple, visualizable explanations with analogies

### 🔍 **3. Detailed Breakdowns**
- Subtopic-by-subtopic deep dives
- Syntax, usage, scoping, and behavior
- Real code examples and comparisons

### ⚠️ **4. Problems & Pitfalls**
- What's allowed vs. disallowed
- Common misconceptions and mistakes
- Production and interview pitfalls

### 💡 **5. Expert Insights**
- Hard-earned lessons from production systems
- When to use vs. avoid, and why
- What seasoned developers watch for

### 🧭 **6. Comparison Tables**
- Side-by-side comparisons with related patterns
- Decision trees for clarity

### 🛠️ **7. Practical Exercises**
- Self-assessment questions
- Debug and rewrite examples
- Realistic problems to apply theory

### ✅ **8. Best Practices**
- Dos and don'ts
- Naming, placement, and structure tips
- Performance and readability considerations

### 🔁 **9. Summary & Cheat Sheet**
- One-page revision summary
- Fast-glance reminders

## 🚀 Quick Start

### Option 1: Use Online (Recommended)
1. Visit [Live Demo](https://naitik-lodhi.github.io/prompt-generator/)
2. Enter your engineering topic
3. Generate and copy your prompt
4. Use with your preferred AI assistant

### Option 2: Run Locally
```bash
# Clone the repository
git clone https://github.com/Naitik-Lodhi/prompt-generator.git

# Navigate to the project
cd prompt-generator

# Open in your browser
open dynamic_prompt_generator.html
# or
start dynamic_prompt_generator.html
```
